# Personal Web Page of Juan Carlos
A mini-project to demonstrate html, css and javascript.

## Features  
- Background color animation in CSS.
- Responsive design for landscape and portrait view, to ensure site will look good on desktop and mobile.
- Javascript onclick effects to change section text and images.

To view the webage, open browser and type in the url: https://jcgreyes.github.io/cmsc207-miniproject/
